# aniscraping
A simple bash script to watch anime by taking nyya torrents and playing with peerflix tool

## Dependencies
1. mpv o vlc o mplayer **Default is MPV**
2. peerflix
3. wget y curl

## Using VLC
Just change the player variable 'mpv' to player='vlc' of the script
